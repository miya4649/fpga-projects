FPGA Synthesizer Project

http://cellspe.matrix.jp/zerofpga/pictheremin.html

* Target FPGA boards

Terasic DE0-CV

* How to build

Development Environment: Ubuntu 16.04 later, Quartus Prime 15.0 later

Install Packages

sudo apt-get install curl unzip git build-essential manpages-dev openjdk-8-jdk ant

Install Synthesijer

http://synthesijer.github.io/web/

On a terminal,

cd pictheremin/synthesijer

make

On Quartus Prime,

Open Project: pictheremin/de0-cv/de0_cv_start.qpf
